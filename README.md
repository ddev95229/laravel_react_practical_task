Please follow the instructions for setting up this project:

BACKEND SETUP:

```
cd my-laravel-app
```

```
composer install
```

```
php artisan migrate --seed
```

```
php artisan key:generate
```

```
php artisan serve --host=localhost --port=8000
```

FRONTEND SETUP:

```
cd react
```

```
npm install
```

```
npm run dev
```
